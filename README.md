## Sobre o Projeto

O **mybiblioteca** é um sistema web desenvolvido em **Django** para **gerenciamento pessoal de livros**. Ele foi criado para ajudar você a organizar sua coleção particular, mantendo um catálogo digital com todos os detalhes das suas obras favoritas.

### 🎯 Funcionalidades

- **📖 Catálogo Pessoal**
  - Cadastro de livros com título, autor, ISBN, editora e ano
  - Upload de capa para cada livro
  - Visualização da sua coleção completa
  - Busca rápida por título ou autor

## 🛠️ Tecnologias Utilizadas

- **Backend**: Python 3.12+ com Django 6.0+
- **Frontend**: HTML5, CSS3 , Bootstrap 5
- **Banco de Dados**: SQLite3 (leve e ideal para uso pessoal)
- **Gerenciador de Pacotes**: UV
